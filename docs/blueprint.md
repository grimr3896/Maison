# **App Name**: MaisonAI

## Core Features:

- Product Catalog: Display Maison's product catalog with categories for kitchen, furniture, and gaming.
- Category Browsing: Enable users to browse products by category (kitchen, furniture, gaming) with sub-groupings within each category.
- Product Details Pages: Provide detailed information for each product, including name, model, description, features, and specifications.
- Inquiry Form: Allow users to submit an inquiry about a product. This form collects name, phone number, email, and a message.
- Smart Recommendations: AI-powered tool that suggests related products or complementary items based on the currently viewed product.

## Style Guidelines:

- Primary color: Warm and inviting tan (#D4A373), evoking a sense of home and sophistication.
- Background color: Soft cream (#F2EAD3), providing a neutral and elegant backdrop.
- Accent color: Muted brown (#A9927D), complementing the primary color and adding depth.
- Headline font: 'Playfair', a modern sans-serif that conveys an elegant and high-end feel.
- Body font: 'PT Sans', a humanist sans-serif, ensuring readability and warmth.
- Use minimalist icons in a linear style to represent categories and features, maintaining a clean and modern aesthetic.
- Maintain a clean and spacious layout with generous white space to enhance readability and highlight products.
